# Module 03 Git & GitHub Fundamentals

## Why do I need to know this?

When I first started learning, I thought Git and GitHub were basically the same things or had similar function. They are not. They are actually two different tools that work together. 

So what are actually Git and GitHub?

Git is a Version Control System (VCS). It allows us to record every change we make to a project, experiment safely, and go back to previous versions if something goes wrong. 

GitHub, on the other hand, is an online platform that hosts Git repositories. It allows us to store our projects in the cloud, synchronize our work across different computers, collaborate and share our work with others. 

Think Git as a time machine that records the history of our project locally while GitHub is like an online platforms (social media) or a logbook, where we store, document, back up, and share our projects. 

Learning Git and GitHub is essential because they help us to organize our work, protect our progress, and build a professional portfolio for our projects.
